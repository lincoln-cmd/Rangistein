# Rangistein 콘텐츠 작성 규칙

## 기본 원칙

- 과학 사실과 개인 의견/해석을 구분해서 작성한다.
- 가능하면 참고 자료를 함께 남긴다.
- 제목은 짧고 명확하게 작성한다.
- 설명(description)은 1~2문장으로 요약한다.

## 카테고리 사용 기준

- 수학: 수학 이론, 개념 정리, 문제 풀이 방향
- 물리: 물리학 개념, 최신 연구 동향, 기초 이론
- 화학: 화학 결합, 분자 구조, 화학 기초
- 생명과학: 생명과학 개념, 기초 생물학
- 지구과학: 지질, 기상, 우주 관련 기초 개념
- 천문학: 천문 관측, 우주론, 외계행성, 최신 연구
- 개인 공부 노트: 복습 기록, 학습 메모, 정리 노트

## 태그 사용 기준

- 너무 세분화하지 않는다.
- 같은 의미의 태그는 하나로 통일한다.
- 우선순위 예시:
  - 기초 개념
  - 최신 동향
  - 논문 요약
  - 공부 노트
  - 물리학
  - 천문학

## 게시 전 체크리스트

- draft 여부 확인
- category / categorySlug 일치 확인
- description 작성 여부 확인
- references 필요 시 추가
- 태그 과다 사용 여부 확인

## 시각화 사용 기준

- 표, 막대 차트, 선 차트는 본문 안에서 `{{viz:...}}` 마커로 삽입한다.
- 마커의 ID는 visualizations 블록의 `id`와 정확히 일치해야 한다.
- 표 헤더와 차트 라벨은 쉼표로 구분한 한 줄 문자열로 입력한다.
- 표 행도 쉼표로 구분한 문자열 한 줄을 한 행으로 입력한다.
- 차트 값 목록은 숫자를 쉼표로 구분해서 입력한다.
- 복잡한 그래프는 V1 범위 밖이므로, 필요 시 이미지로 먼저 대체한다.
