# 랑이슈타인 출시 직전 체크리스트

## 1. 콘텐츠 공개 상태 확인

- [ ] 테스트용 게시물 공개 여부 확인
- [ ] 제목, 설명, 태그가 비어 있는 게시물이 없는지 확인
- [ ] draft 상태 게시물이 의도대로 처리되는지 확인
- [ ] 공개하기 곤란한 임시 문구, 메모, 테스트 텍스트가 남아 있지 않은지 확인
- [ ] 대표 글 설정이 의도한 게시물에만 적용되어 있는지 확인

## 11. 출시 후 첫 운영 원칙

- 새 글 1~3개 먼저 안정적으로 발행해 본다.
- 발행 직후 홈, 카테고리, 태그, 검색 반영 상태를 확인한다.
- 운영 중 발견되는 불편함은 대규모 개편보다 작은 수정으로 우선 대응한다.
- 기능 추가보다 콘텐츠 축적을 우선한다.
