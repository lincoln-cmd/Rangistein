# 랑이슈타인 (Rangistein)

과학 이론, 최신 동향, 논문 요약, 개인 학습 기록을 정리하는 개인 과학 웹 아카이브입니다.

## 프로젝트 개요

랑이슈타인은 수학, 물리, 화학, 생명과학, 지구과학, 천문학을 중심으로 과학 콘텐츠를 정리하는 개인 웹 사이트입니다.

이 프로젝트는 단순한 링크 모음형 블로그가 아니라 다음 목적을 함께 지향합니다.

- 과학 관련 지식을 체계적으로 정리하는 개인 아카이브
- 개인 학습 기록과 해설을 함께 제공하는 지식형 웹 사이트
- 기획, 개발, 운영 역량을 보여줄 수 있는 포트폴리오
- Git 기반 콘텐츠 관리와 정적 배포를 결합한 지속 가능한 개인 프로젝트

## 현재 운영 상태

현재 프로젝트는 다음 흐름으로 실제 운영 가능한 상태입니다.

- Decap CMS에서 글 작성 및 수정
- GitHub 저장소에 반영
- Cloudflare Pages 자동 배포
- 배포 사이트에서 최종 확인

로컬에서 CMS 저장 직후 본문이 즉시 반영되는 구조는 사용하지 않습니다.
운영 기본 흐름은 다음과 같습니다.

CMS → GitHub → Cloudflare Pages → 배포 사이트 확인

## 관리자 운영

관리자 페이지 경로:

    /admin/index.html

운영 방식:

- 방문자 로그인 기능은 제공하지 않음
- 운영자 1인 기준 관리 구조
- Decap CMS에서 게시글 작성 및 수정
- Git 저장소에 변경 이력 기록
- GitHub 반영 후 Cloudflare Pages 자동 배포

## 콘텐츠 운영 원칙

- 가능한 한 출처와 참고 자료를 함께 정리합니다.
- 논문 및 기사 내용은 원문 복제가 아니라 요약과 해설 중심으로 작성합니다.
- 확인된 사실과 개인 의견 또는 해석을 구분해 작성합니다.
- 학습 기록은 개인 공부 노트 성격임을 명확히 구분합니다.
- 최신 동향 콘텐츠에는 작성일 또는 수정일을 표시합니다.
