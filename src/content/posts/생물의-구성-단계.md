## 생물의 특성
- 세포로 구성되어 있다. 생물은 생명 활동의 기본 단위인 세포로 이루어져 있다.
- 물질대사를 한다. 생물은 외부에서 물질을 받아들이고, 이를 이용해 에너지를 얻거나 몸을 구성하는 물질을 합성한다.
- 자극에 반응한다. 생물은 빛, 온도, 소리, 화학 물질 등의 자극을 받아들이고 그에 알맞게 반응한다.
- 항상성을 유지한다. 생물은 외부 환경이 변해도 몸속 상태를 일정하게 유지하려는 성질을 가진다.
  - 예: 체온 조절, 혈당량 조절, 삼투압 조절
- 발생과 성장을 한다. 생물은 수정란 또는 하나의 세포에서 시작해 세포 분열과 분화를 거쳐 몸을 형성하고 성장한다.
- 생식과 유전을 한다. 생물은 자신과 닮은 자손을 만들며, 유전 정보를 다음 세대로 전달한다.
- 적응과 진화를 한다. 생물은 환경에 적응하며, 오랜 시간에 걸쳐 집단의 유전적 특성이 변하는 진화를 겪는다.

## 다세포생물의 구성 단계
: 다세포 생물은 여러 세포가 모여 몸을 이루며, 세포들이 기능에 따라 조직, 기관 등으로 단계적으로 구성된다.
### 동물의 구성 단계
- 세포 → 조직 → 기관 → 기관계 → 개체
  - 세포: 

     ![동물 세포](/uploads/동물 세포.jpg "동물 세포")

    - 생명 활동의 기본 단위이다. 세포막, 핵, 미토콘드리아, 리보솜, 소포체, 골지체 등의 세포 구조를 가진다.

  - 조직
    - 모양과 기능이 비슷한 세포들이 모여 특정한 기능을 수행하는 단계이다.
    - 예: 상피 조직, 결합 조직, 근육 조직, 신경 조직


  - 기관
    - 여러 조직이 모여 일정한 기능을 수행하는 구조이다.
    - 예: 뇌, 척수, 위, 작은창자, 폐, 기관지, 콩팥, 방광


  - 기관계
    - 관련된 기능을 수행하는 여러 기관이 모여 이루어진 체계이다.
    - 예: 신경계, 소화계, 호흡계, 배설계, 순환계

  - 개체
    - 여러 기관계가 통합되어 하나의 독립된 생명체를 이룬 것이다.


### 식물의 구성 단계
- 세포 → 조직 → 조직계 → 기관 → 개체
  - 세포

    ![식물 세포](/uploads/식물 세포.jpg "식물 세포")

    - 생명 활동의 기본 단위이다. 식물 세포는 세포막, 핵, 미토콘드리아, 리보솜, 소포체, 골지체 외에도 세포벽, 엽록체, 액포 등을 가진다.

  - 조직
    - 비슷한 기능을 하는 세포들이 모인 단계이다.
    - 예: 표피조직, 유조직, 기계조직, 통도조직

  - 조직계
    - 여러 조직이 모여 식물체의 일정한 기능을 담당하는 단계이다.
    - 예: 표피 조직계, 기본 조직계, 관다발 조직계
      - 표피 조직계: 표피조직
      - 기본 조직계: 유조직, 기계조직
      - 관다발 조직계: 통도조직

  - 기관
    - 여러 조직계가 모여 특정한 기능을 수행하는 식물의 구조이다.
    - 예: 뿌리, 줄기, 잎, 꽃, 열매

  - 개체
    - 여러 기관이 모여 하나의 독립된 식물체를 이룬 것이다.

## 생명 시스템의 구성 단계
- 세포 → 조직 → 기관 → 개체 → 개체군 → 군집 → 생태계
- 개체군: 일정한 지역에서 같은 종의 개체들이 무리를 이루어 살아가는 집단
- 군집: 일정한 지역에 모여 살아가는 여러 종류의 개체군들의 집단
- 생태계: 생물이 다른 생물 및 주위 환경과 서로 영향을 주고받으며 생활하는 통합된 시스템. 생물은 대부분 무리를 이루어 살아가며, 다른 무리의 생물뿐만 아니라 빛, 온도, 물, 토양, 공기 등의 환경과 영향을 주고받으며 살아간다.
